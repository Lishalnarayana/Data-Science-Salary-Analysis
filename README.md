# 📊 Data Science Salary Analysis

## Project Overview

This project analyzes global data science salaries from 2020–2022 to identify trends in compensation across different experience levels, job roles, company sizes, work arrangements, and locations.

Using Python for data cleaning, exploratory data analysis (EDA), visualization, and machine learning, the project provides insights into the factors that influence salaries in the data science industry.

---

## Dataset

- **Dataset:** Data Science Salaries (2020–2022)
- Contains salary information for data science professionals worldwide.

---

## Business Questions

This project answers the following questions:

1. Does experience level influence salary?
2. Do larger companies pay higher salaries?
3. Does remote work affect salary?
4. Which data science roles have the highest average salary?
5. How have salaries changed between 2020 and 2022?

---

## Machine Learning

A Linear Regression model was developed to predict salary based on selected job-related features.

Model workflow included:

- Data preprocessing
- Feature encoding
- Train-test split
- Model training
- Performance evaluation using regression metrics

## Key Insights

- Senior and Executive professionals earn significantly higher salaries than Entry-level employees.
- Company size influences average compensation.
- Remote work offers competitive salaries across many job roles.
- Specialized data science positions generally receive higher salaries.
- Average salaries increased steadily between 2020 and 2022.

---

## Results

The analysis found that salaries increase consistently with professional experience, with Executive-level professionals earning the highest average compensation. The project also identified trends across company sizes, remote work, and job roles, and used a Linear Regression model to predict salaries based on job-related features.
